->> How to Run the Project

Download or clone this repository.

Place the project folder inside your server directory:

For XAMPP → htdocs/

For WAMP → www/

Start Apache and MySQL.

Import the SQL file into phpMyAdmin (if database is included).

Run the project in your browser:http://localhost/project-folder-name/
